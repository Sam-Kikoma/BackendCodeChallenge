# Backend Code Challenge

This project reads products from `feed.xml`, splits them into batches and prints each batch through a mock external service.

## Prerequisites

- Node.js 18 or later

## Install

````bash
npm install

## Run in development
```bash
npm run dev

## Build and run
```bash
npm run build
npm start

## Notes
- The input file is `feed.xml` in the project root.
- Output is written to the console.
````
